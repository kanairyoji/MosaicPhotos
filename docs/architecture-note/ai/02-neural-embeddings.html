2. ニューラルネットと埋め込み

ニューラルネットは「重みを調整して入力から有用な特徴を取り出す関数」。その途中で得られる意味の数値表現＝埋め込みが、検索の主役になる。

ニューラルネットの超概要

入力（画素や単語）に重みを掛けて足し、非線形関数を通す——これを何層も重ねる。学習とは、正解との誤差が小さくなるよう重みを少しずつ更新すること（誤差逆伝播）。層を深くすると、低レベル特徴（エッジ）から高レベル特徴（物体・概念）へと段階的に抽象化される。

graph LR IN[入力] --> L1[層1
低レベル特徴] L1 --> L2[層2
中レベル] L2 --> L3[層3
高レベル概念] L3 --> OUT[出力 / 埋め込み]

ベクトルとは

単に数値の並び。たとえば [0.12, -0.98, 0.34, ...]。AI では「意味の座標」として使う。次元数（要素数）が大きいほど多くの意味を表せる（CLIP は 512 次元が典型）。

埋め込み = 意味の座標

モデルの最終層付近で得られるベクトルを埋め込みと呼ぶ。よく訓練されたモデルでは、似た意味のものは近くに、違うものは遠くに配置される。これが「ベクトルの近さ＝意味の近さ」という検索の前提を生む。

graph TD A["犬の写真"] --> P1["(0.8, 0.1)"] B["子犬の写真"] --> P2["(0.79, 0.12)"] C["車の写真"] --> P3["(-0.6, 0.7)"] P1 --- N[近い = 似た意味] P2 --- N P3 --- F[遠い = 異なる意味]

類似度の測り方

指標性質
ユークリッド距離2 点間の直線距離。小さいほど近い。
内積 (dot)大きいほど同方向。長さの影響を受ける。
コサイン類似向きだけを見る（長さ非依存）。埋め込み比較の定番。-1〜1。

多くの埋め込みは長さを 1 に正規化して使うため、コサイン類似と内積が一致する。詳しくは第5章。

表現学習という発想

分類や検索のたびに人手で特徴を設計するのではなく、モデルに良い表現（埋め込み）を学ばせるのが現代の主流。一度よい埋め込みが得られれば、検索・分類・推薦・クラスタリングなど多用途に流用できる。

要点 「埋め込み = 意味の座標」「近さ = 意味の近さ」。この 2 つを押さえれば、以降の章（CLIP・ベクトル検索）はすべてこの応用として理解できる。
