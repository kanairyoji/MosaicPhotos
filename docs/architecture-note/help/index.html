MosaicPhotos ヘルプ

端末内の写真と Dropbox の写真を、ひとつのアプリでまとめて見られる写真ビューワーです。 使い方をトピック別にまとめています。

フォルダアルバムの作り方（正規表現ガイド）

Dropbox のフォルダ名から自動でアルバムを作る方法。コピペで使える例をたくさん載せています。

はじめに・基本の見かた準備中

ホーム画面、All / Photos / Cloud の切り替え、グリッドのズームと全画面表示。

AI アルバムと意味検索準備中

自然文でアルバムを作る・写真を探す（オンデバイス CLIP）。

場所アルバム準備中

市区町村ごとの自動グループ表示。

Dropbox 連携とバックアップ準備中

接続、同期、端末写真の Dropbox へのバックアップ。

設定（電源・通信・キャッシュ・表示）準備中

バックグラウンド処理、グリッド密度、ストレージ、言語など。

困ったとき準備中

よくある質問とトラブルシューティング。
