用語集

本資料に出る用語の短い定義。

AI / 検索

用語意味
CLIP画像とテキストを同じベクトル空間に写すモデル。本アプリは MobileCLIP-S2 を Core ML 化。
埋め込み (embedding)意味を表す固定長の数値ベクトル（本アプリは 512 次元）。
コサイン類似2 ベクトルの向きの近さ（-1〜1）。埋め込みの一致度に使う。
オープン語彙固定リストに縛られず任意の文で検索できること。
ゼロショット再学習なしで候補ラベルとの近さで分類する手法。表示タグに使用。
RRFReciprocal Rank Fusion。複数のランキングを順位で統合する手法。
Foundation ModelsOS 内蔵のオンデバイス LLM。本アプリではクエリ英訳に使用。

永続化 / 並行性

用語意味
SwiftDataApple の永続化フレームワーク。@Model / ModelContext / ModelContainer。
@ModelActorModelContext を専用アクタに束縛して安全に並行アクセスする属性。
@ObservableSwift Observation の状態観測属性。参照プロパティ単位で再描画。
@MainActorメインスレッドに隔離する属性。本アプリのアプリターゲットは既定 MainActor。
jetsamiOS がメモリ不足時にプロセスを強制終了する仕組み。
watchdog起動/応答が遅すぎるアプリを OS が終了させる監視。

本アプリ固有の型

用語意味
PhotoStore / PhotoItem写真ソースを統一する表示プロトコル（PhotoSourceKit）。
MergedPhotoStoreローカル + Dropbox を統合する PhotoStore。
EnrichedPhoto付加情報済みの 1 枚を表す Sendable 値型。
PhotoEnrichment / PhotoEmbeddingメタデータ / CLIP 埋め込み（fp16・別テーブル）の @Model。
refKeyローカル/クラウドを統一する写真キー（"L-…"/"C-…"）。PhotoRef でエンコード。
seam差し替え可能な継ぎ目。本アプリでは知覚・翻訳・ラベラのプロトコル境界。
合成ルート (Composition Root)依存を結線する一点。本アプリは AutoAlbumAdapters など。
