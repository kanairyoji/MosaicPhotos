場所アルバム

位置情報のある写真を、市区町村ごとに自動でまとめて表示します。端末と Dropbox の 写真を一緒に扱います。

見かた

  • ホーム下部の 「場所（Places）」 から、市区町村ごとのまとまりを開けます。
  • 表示されるのは位置情報（GPS）を持つ写真だけです。位置情報のない写真はここには出ません。
  • 写真が増えたり位置情報が読み取れたりするたびに、自動で増えていきます。

仕組み

  • 写真の座標から地名（市区町村）を、端末に同梱した地名データベースで完全にオフライン逆引きします （通信は使いません）。一度調べた地名はキャッシュされます。地名は表示言語に合わせて日本語／英語で出ます。
  • 「旅行（時間と場所）」が期間＋訪問地でまとめるのに対し、こちらは場所そのものでまとめます。

うまく出ないとき

  • 写真に位置情報があるかを確認してください（撮影時に位置情報オフだと付きません）。
  • 反映には少し時間がかかることがあります。設定 → Places から、まとめ方の細かさ（おおよその範囲）や 再スキャンの間隔を調整できます。
