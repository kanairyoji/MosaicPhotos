Background & Battery（電源・通信）

バックグラウンドで進む処理（AI の取り込み、自動アルバム生成、Dropbox 同期、バックアップなど）を、 電源と通信の条件でゲートして、電池とモバイル通信を節約します。

  • 電源：充電中のみ（既定）/ 常に / オフ。
  • 通信：Wi-Fi のみ（既定）/ セルラーも許可 など。
いま見ている・開いている写真は常に取得されます。制限されるのは“自動の裏側の処理”だけなので、 閲覧が止まることはありません。AI の取り込みは賢く、セルラー時は端末写真を進めつつクラウド写真は Wi-Fi まで待ちます。

Photo Grid（表示の密度）

月グループ表示の詰め具合を選べます（細：1 行 / ふつう：3 行 / 粗：5 行）。 値を大きくすると日付の見出しが減り、より密に表示されます。詳しくは 基本の見かた。

Storage（キャッシュ）

  • サムネイル・本体画像のキャッシュ容量の上限を設定できます。
  • 大規模なライブラリでも安定するよう、メモリは状況に応じて自動調整されます。

その他

  • 言語：System / 日本語 / English を切り替えられます。
  • Licenses：本アプリと同梱物のライセンス表記。
  • ヘルプ：このオンラインヘルプを開きます。
